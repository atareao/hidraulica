An extension that adds functions for Calc for hydraulic calculations.

REYNOLDS calculates the Reynolds number.
REYNOLDSAGUADULCE calculates the Reynolds number for freshwater.
REYNOLDSAGUADEMAR calculates the Reynolds number for seawater.
VISCOSIDADDINAMICAAGUA calculates the dynamic viscosity of water.
DENSIDADAGUA calculates the density of water.
FRICTION calculated friction in a pipe.
FRICCIONAGUADEMAR calculates the friction of seawater in a pipe.
FRICCIONAGUADULCE calculates the friction of fresh water in a pipe.
PERDIDACARGA calculates the pressure loss in a pipe.
VELOCIDAD calculates the velocity of a fluid in a pipe.
